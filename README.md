# F.CS301 Алгоритмын зохиомж ба шинжилгээ

## Installation

How to install:

```bash
  1. cd client
     npm install
```

```bash
  2. cd
     cd server
     Linux: source venv/bin/activate
     Windows: ./venv/Scripts/activate
     pip install -r requirements.txt
     python main.py
```

```bash
  3. cd
     cd client
     npm run dev
```
